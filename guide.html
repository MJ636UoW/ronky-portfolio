1. Editing Elements via WordPress Customizer

The primary method for updating text fields, section images, and links across the theme is the built-in WordPress Customizer.

1
Access the Customizer Panel

Log in to your WordPress dashboard, and navigate to Appearance > Customize on the left-hand sidebar menu.

2
Change Hero Section Elements

Go to the Hero Section Settings panel:

  • Hero Title: The main page title. You can include HTML tags like <br> to create line breaks, or <span class="accent-text text-glow">WORD</span> to apply the neon glow.
  • Hero Subtitle: Enter the paragraph description displayed underneath the title.
  • Hero Showcase Image: Upload the large editorial background image displayed on the right.
  • Floating Tag Texts (1-5): Change the texts for the 5 floating anti-gravity badges (e.g. Photographer, Videographer).
  • Floating Elements Speed Multiplier: Edit the overall speed multiplier for all 5 floating tags (e.g. 1.0 for default, 1.5 for faster, 0.5 for slower).
3
Change About Section Elements

Go to the About Section Settings panel:

  • About Title & Text: Edit your primary bio/description.
  • Profile Image: Upload the photo displayed inside the glass container.
  • Stats Counters: Edit the numerical targets for your achievements (e.g. 150 projects, 98% satisfaction, etc.). The numbers automatically animate when visitors scroll to this section.
4
Change Services & Contact Settings

Go to Services Settings or Contact Settings:

  • Services (1-4): Change title names, descriptions, and the comma-separated URLs of the media displaying inside the immersive slide-out lightboxes.
  • Contact Details: Enter your target Email Address, WhatsApp phone link, and Instagram handle to dynamically update all action cards.
  • Fluent Form ID: Connect your customized Fluent Form database submission table.

2. Editing the Header Logo & Navigation Menu

Both the header brand logo and the navigation links are fully dynamic and can be configured through standard WordPress settings.

A. Editing the Brand Logo Text

1
Go to Site Identity

Open the Customizer (Appearance > Customize) and select the Site Identity panel.

2
Edit Site Title

Update the Site Title text (e.g., change it to RONKY EDITS, SANDY WEB, or MY BRAND). The theme's code will automatically detect the name, split the words, and insert the styled neon separator dot (•) in place of spaces!

B. Customizing Navigation Links

1
Create your Custom Menu

In your WordPress dashboard, navigate to Appearance > Menus. Click **create a new menu**, add your target pages or custom anchor links (e.g., #about, #portfolio, #services, #contact), and save it.

2
Assign to Primary Menu Location

Under Menu Settings (at the bottom of the menu editor) or under the Manage Locations tab, check the box to assign this menu to the Primary Menu location.

Note: If no custom menu is assigned, the theme automatically displays the default layout links (About, Works, Services, Reviews, Book) as a fallback.

3. Editing & Customizing Elements via Elementor

For pages built with Elementor, you can edit visual styles, change layout spacings, and modify texts directly inside the Elementor drag-and-drop editor.

1
Open the Elementor Editor

Go to Pages in your WordPress dashboard. Hover over the page you wish to modify and click Edit with Elementor.

2
Editing Hero Split & Title Blocks

Hover over the hero content columns, click on the **Edit Genz Premium Hero Split** widget (blue pencil icon). In the left sidebar:

  • Hero Title & Subtitle: Edit text lines inside the text fields. You can adjust font styles or sizes under the Style tab.
  • Hero Image: Go to the Style tab in the left sidebar, click on the image box, and select a new image from your Media Library.
3
Customizing Floating Tags (Floating Badges)

Click on any of the floating tag elements (e.g., Photographer, Videographer). In the left sidebar panel, you can edit the text tag and configure custom depth/speed settings to change the anti-gravity float speed.

4
Modifying Portfolio Grid & Masonry Layouts

Click on the **Selected Works** grid section. In the left panel, you can adjust:

  • Limit: Change how many posts display on the page before pagination.
  • Filters: Select which categories (e.g. Fashion, Editorial) are active for the buttons.
  • Spacings & Columns: Adjust card padding values under the Style tab.
5
Editing Client Review Slides

Click on the Testimonial Slider element to modify display options (e.g., autoplay speeds, navigation arrow sizes) or edit the layout columns.

4. Changing Brand Colors, CSS Themes, & Typography

You can change the brand color scheme (e.g., change the signature neon green to red, gold, or purple) and customize overall typography by editing global CSS variables.

A. Locate Theme Stylesheets

  • WordPress Theme CSS: [theme-root-folder]/assets/css/theme-style.css
  • Static Site CSS: src/style.css (used to compile build for static hosting like Vercel)

B. Changing Color Variables

Open the stylesheet and locate the :root selector at the top. Modify these hex color codes to change the website colors:

:root {

--bg-primary: #0A0A0A; /* Page background color */

--bg-secondary: #121212; /* Containers background color */

--accent: #B8FF00; /* Main brand neon green color */

--accent-glow: rgba(184, 255, 0, 0.35); /* Neon glow effect color */

}

To change to a Hot Neon Pink theme, update --accent to #FF007F and --accent-glow to rgba(255, 0, 127, 0.35).

C. Changing Typography Fonts

Change the font-family properties under the :root configuration:

  • Body Fonts: Edit --font-sans (default is 'Plus Jakarta Sans').
  • Header Display Fonts: Edit --font-display (default is 'Syne').

5. Adding and Categorizing Portfolio Works

Portfolio items display dynamically on your site using standard WordPress Posts.

1
Add a New Work

Go to your WordPress Admin panel and navigate to Posts > Add New. Enter your work title.

2
Select the Category

Under the **Categories** panel on the right sidebar, assign one or more categories:

  • Fashion: Displays under the FASHION filter tab.
  • Editorial: Displays under the EDITORIAL filter tab.
  • Video: Displays under the VIDEO filter tab and automatically configures video playback.
3
Set Featured Image (Thumbnail)

Under the **Featured Image** panel on the right, click **Set featured image** and upload the showcase thumbnail for this work. If left empty, the card will automatically cycle through standard layout placeholders.

4
Configure Hover Video Playback (Optional)

For video posts, scroll down to the **Custom Fields** panel. Click **Add Custom Field**, name the key exactly video_url, and paste the URL of your MP4 file. The card will then play the video preview on hover.

6. Adding Dynamic Testimonials

Reviews are managed dynamically via the dedicated Testimonials custom post type.

1
Access the Testimonials Menu

Click on the **Testimonials** menu item (speech bubble icon) in your WordPress left sidebar.

2
Create a Review Card

Click **Add New Testimonial** and enter:

  • Title Box: The client's name (e.g. Sofia Laurent).
  • Editor Content Box: Write the client's review quotation.
  • Featured Image: Upload the client's profile photo or logo avatar.
  • Client Details: Enter their job title or company in the bottom custom meta box (e.g. Creative Lead, MAISON Agency).
3
Publish

Click **Publish**. The home page reviews slider will automatically update and loop your new testimonial.

7. Customizing and Connecting Fluent Forms

The contact form is built on Fluent Forms, integrated with custom AJAX submission handlers.

1
Modify Form Fields

In your WordPress sidebar, go to Fluent Forms. Click **Edit** next to the form titled Ronky Edits Portfolio Form. You can drag and drop fields to add, edit, or remove entries.

2
Find the Form ID

Note down the ID of the form from the Fluent Forms list (e.g. if the shortcode is [fluentform id="1"], the ID is 1).

3
Sync Form ID with Theme Customizer

Go to **Appearance > Customize > Contact Settings**. Under the **Fluent Form ID** field, type your form ID and click **Publish** to connect your form to the theme submission handler.

8. Step-by-Step Live Hosting & Migration Guide

To migrate the website from your local development environment to a live web hosting server (like Hostinger, Bluehost, or SiteGround), follow these steps:

Part A: Migrating the WordPress Website

1
Export Local Site Data

Log in to your local WordPress dashboard. Go to Plugins > Add New, search for All-in-One WP Migration, install and activate it. Navigate to **All-in-One WP Migration > Export**, select **Export To > File**, and download the compiled backup file (.wpress) to your computer.

2
Install WordPress on Live Server

Log in to your live hosting panel (e.g., Hostinger hPanel). Use the Autoinstaller to set up a fresh WordPress instance on your target domain name.

3
Import Site Data Live

Log in to your live domain's WordPress admin dashboard (e.g., yourdomain.com/wp-admin/). Go to Plugins, install and activate All-in-One WP Migration. Go to Import > Import From > File, select your local .wpress backup file, and run the import. Once complete, your database, posts, customizer settings, forms, and themes will be fully active on your live server!

Part B: Deploying Static Files to Vercel

  1. Edit files inside your local project folder.
  2. Open your terminal and build static files: npm run build.
  3. Commit the changes to your Git repository:
    • git add .
    • git commit -m "Site updates"
  4. Push the commit to GitHub: git push origin main.
  5. Vercel automatically rebuilds and deploys the new updates.
